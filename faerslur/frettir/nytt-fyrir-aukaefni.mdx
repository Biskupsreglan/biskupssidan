Matthías er búinn að vera mjög upptekinn síðustu daga í því að gefa fólki sem er skráð inn í regluna meira að gera.
Þeir sem geta skráð sig inn í regluna geta séð aukefni, fengið valmöguleikan að raða meðlimum uppi öðruvísi og fleira.

T.d. hérna eru meðlimir flokkaðir eftir meðalvirði (öll stig frá færslum deilt með færslu fjölda).

![Meðlimir flokkaðir eftir meðalvirði](/static/images/faerslur/annad/flokka.png)

Þú getur commentað með github aðgang, þú þarft ekki að vera skráður inni í vefsíðuna fyrir það.
Til að vera skráður inn í vefsíðuna þarf að vera búið að samþykkja github aðganginn þinn.

Sentu email á aukaefni@biskupsreglan.com um hver þú ert og hvaða email github aðgangurinn þinn er
svo það þú getur skráð þig inn.
